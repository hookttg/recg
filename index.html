What is RECG?

RECG is a R package for ECG analysis, created by Mile group (www.milegroup.net). This package is intended to be used by researchers in the area of signal processing, in particular the ECG processing.

RECG is still under construction

This is not a stable version of the package, it is a version still under construction. This means that it may contain errors and that there is some functions that are still under tests, so it may not produce the expected results.

In a few months we expect to release a new (stable) version, correcting the errors and adding more functionality. At this moment the package includes 4 different QRS detection algorithms and a delineation algorithm (in test phase).
